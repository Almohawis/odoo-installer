# odoo installer for ubuntu 24.4
Shell Script to install and setup odoo 18 and odoo 19 With One Press on Ubuntu 24.4 LTS

## How To Use?
 1. git clone https://github.com/Almohawis/odoo-installer.git
 2. chmod +x ./setup-odoo
 3. ./setup-odoo
 4. Select The Version
 5. Confirm
 6. Enter Admin Password
 7. Select The Port " For odoo 18 "
 8. Enjoy (:
